## Descriptions [![CircleCI](https://circleci.com/gh/wszgxa/react-ssr-skeleton.svg?style=svg)](https://circleci.com/gh/wszgxa/react-ssr-skeleton)

A react server side skeleton repo.

## Prerequisites

    node:8.9.1 (nvm use)
    yarn

## DEV
``` bash
yarn install
yarn start:dev
```

## Prod
```bash
yarn install
yarn start:prod
```
